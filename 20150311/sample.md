2015/03/11 勉強会メモ
====

例えば

```
abc => Abc // 頭文字のみ大文字に変換したい
```

下のどちらがオブジェクト指向的か？

```
// 誰かがやる
// Java
StringUtil.headToUpper("abc"); => Abc
```

```
// 自分でやる
// Ruby(Mixin, method missing), Scala(implicit, Macro)
"abc".headToUpper(); => Abc
```
